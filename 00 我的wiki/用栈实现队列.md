### 用栈实现队列



要实现的队列的API如下：

```c++
class MyQueue {
public:
    MyQueue() {}
    void push(int x); /* 添加元素到队尾 */ 
    int  pop(); 	  /* 删除队首元素,并返回该元素值 */
    int  peek(); 	  /* 返回队首元素，不会删除该元素 */
    bool empty();	  /* 判断队列是否为空 */
};
```



C++中的栈容器具有以下操作：

```c++
std::stack<int> s;
void push(int x); // 添加元素到栈顶
void pop();   	  // 删除栈顶元素,但不返回该元素值
int  top(); 	  // 返回栈顶元素,但不将该元素弹出栈
bool empty();     // 判断栈是否为空
```



使用2个栈实现1个队列如下：

```c++
class MyQueue {
private:
    std::stack<int> s1;
    std::stack<int> s2;
public:
    MyQueue() {}

    void push(int x) {
        s1.push(x);
    }

    int pop() {
        int front = peek(); // 调用peek保证s2非空
        s2.pop();
        return front;
    }

    int peek() {
        // 从s1中弹出元素依次压入s2中,若s2中非空则每次弹出元素即可作为队首元素
        if (s2.empty()) {
            while (!s1.empty()) {
                s2.push(s1.top());
                s1.pop();
            }
        }
        return s2.top();
    }

    bool empty() {
        return s1.empty() &&  s2.empty();
    }
};
```
